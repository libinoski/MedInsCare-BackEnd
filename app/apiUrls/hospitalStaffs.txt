
1. **Authentication and Profile Management:**
   - Login route for hospital staff: `POST http://localhost:1313/api/mic/hospitalStaff/hospitalStaffLogin`
   - Change password route for hospital staff: `POST http://localhost:1313/api/mic/hospitalStaff/hospitalStaffChangePassword`
   - Change ID proof image route for hospital staff: `POST http://localhost:1313/api/mic/hospitalStaff/hospitalStaffChangeIdProofImage`
   - Change profile image route for hospital staff: `POST http://localhost:1313/api/mic/hospitalStaff/hospitalStaffChangeProfileImage`
   - View profile route for hospital staff: `POST http://localhost:1313/api/mic/hospitalStaff/hospitalStaffViewProfile`
   - Update profile route for hospital staff: `POST http://localhost:1313/api/mic/hospitalStaff/hospitalStaffUpdateProfile`

2. **Hospital News Management:**
   - View all hospital news route for hospital staff: `POST http://localhost:1313/api/mic/hospitalStaff/hospitalStaffViewAllHospitalNews`
   - View one hospital news item route for hospital staff: `POST http://localhost:1313/api/mic/hospitalStaff/hospitalStaffViewOneHospitalNews`

3. **Notifications Management:**
   - View all notifications from hospital route for hospital staff: `POST http://localhost:1313/api/mic/hospitalStaff/hospitalStaffViewAllNotifications`
   - View one notification from hospital route for hospital staff: `POST http://localhost:1313/api/mic/hospitalStaff/hospitalStaffViewOneNotification`

4. **Patient Management:**
   - Register patient route: `POST http://localhost:1313/api/mic/hospitalStaff/registerPatient`
   - View all patients route for hospital staff: `POST http://localhost:1313/api/mic/hospitalStaff/hospitalStaffViewAllPatients`
   - View one patient route for hospital staff: `POST http://localhost:1313/api/mic/hospitalStaff/hospitalStaffViewOnePatient`
   - Search patients route for hospital staff: `POST http://localhost:1313/api/mic/hospitalStaff/hospitalStaffSearchPatient`

5. **Medical Records and Discharge Management:**
   - Add medical record route for hospital staff: `POST http://localhost:1313/api/mic/hospitalStaff/hospitalStaffAddMedicalRecord`
   - Request discharge of one patient route for hospital staff: `POST http://localhost:1313/api/mic/hospitalStaff/hospitalStaffRequestDischarge`
   - View all approved discharge requests route for hospital staff: `POST http://localhost:1313/api/mic/hospitalStaff/hospitalStaffViewAllApprovedDischargeRequests`
